lattepanda安装ubuntu18.04全教程

教程参考lattepanda官方教程

下载ubuntu18.04 iso

ubuntu18.04下载，下载desktop，server会出现bios反复横跳加载配置，显示让重启，重启后无操作系统。

制作启动盘

准备

  1. u盘
  2. 下载好的镜像
  3. Rufus

制作步骤

  1. 将u盘格式化

  2. 安装制作驱动盘

    ==注意：分区类型必须调整为GPT==

  3. 点击开始，按照提示安装缺少的依赖。制作完成。

BIOS配置

开机

长按电源键，至蓝色引导灯亮起，多次敲击ESC键，进入bios

bios配置

CSM Support [Disable]

UEFI PXE Boot [Ipv4 Only]

Legacy PXE Boot [Disabled]

Force Boot from Windows Boot [Enabled]

将Boot Option Priorities下的 Boot Option #1 改为[ubuntuxxxxx]

选择Save changes and Reset

进入安装界面

安装

There will be two options:

  • Try Ubuntu without installing
  • Install Ubuntu

Both options should work, but in this tutorial we will install Ubuntu. We recommend proceeding with this option.

按照ubuntu的安装引导安装即可。安装完成后拔掉u盘，否则可能导致bios持续引导启动安装，选择重启，出现ubuntu-desktop即为安装成功。
